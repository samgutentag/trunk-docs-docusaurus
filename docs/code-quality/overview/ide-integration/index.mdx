# IDE integrations

Code Quality helps you shorten the feedback loop by integrating with your favorite IDEs and code editors.

### How it works

Code Quality runs [a daemon](../index#daemon) that looks for files that change in real time and lints the changes using the same tools and configuration as running `trunk check`. With LSP support, you will get instant feedback on your code changes as you write.
